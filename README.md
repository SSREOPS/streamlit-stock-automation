# Streamlit Stock Dashboard

This is a Streamlit app for fetching and displaying live stock statistics using Yahoo Finance data.

## How to Run

1. **Clone the repository:**
   ```
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name
   ```

2. **Install dependencies:**
   ```
   pip install -r requirement.txt
   ```
   *(If the filename is `requirements.txt`, use that in the command instead.)*

3. **Run the Streamlit app:**
   ```
   streamlit run SAM_App.py
   ```

## Features

- Enter one or more stock tickers to fetch live statistics
- Download the results as a formatted Excel file
